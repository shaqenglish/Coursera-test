so to access browser sync live editing simply do the following, navigate to dir with index file on cmd, then launch browser sync with the following command browser-sync start --server --directory --files "*" and live editing will be launched, on git first go to main repository file on local machine, then git status, then git add "." then git commit -m "message you would like to add", it will be ready for sharing at this point, then git push to send to github and git pull to sync with local pc, to start with sublime text simply start with html then ctrl + space to add basic tags, then save file as .html to the folder of your choice then steps above apply. keep in mind that if you want to use terminal you can press ctrl + c to stop it it will prompt you if youre sure by typing "y" or "n"
